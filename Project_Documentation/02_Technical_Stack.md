# Technical Stack

This project is built using a modern React ecosystem focused on performance and developer experience.

## Core Framework

- **Runtime**: [React 19](https://react.dev/)
- **Build Tool**: [Vite 7](https://vitejs.dev/)
- **Routing**: [React Router 7](https://reactrouter.com/)

## Styling & UI

- **CSS Framework**: [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Primitives**: [Radix UI](https://www.radix-ui.com/) (Dialog, Dropdown, Collapsible, etc.) - Accessible unstyled components.
- **Icons**:
  - `lucide-react`: Primary icon set.
  - `react-icons`: Supplementary icons.

## Animations & Interactivity

- **Motion (framer-motion successor)**: Declarative animations.
- **GSAP**: Complex timeline-based animations.
- **Rive React Canvas**: Interactive vector animations.
- **Lenis**: Smooth scrolling.

## State & Data

- **Context API**: React's built-in state management (e.g., `AuthContext`).
- **React Hot Toast**: Notifications and toast messages.

## Other Key Libraries

- **Embla Carousel & Swiper**: Carousel/slider functionality.
- **Chart.js**: Data visualization for dashboards.
- **React OAuth Google**: Google authentication support.
